Q7.Is magic?

Description

Given a number A, check if it is a magic number or not.

A number is said to be a magic number if the sum of its digits is calculated till a single digit recursively by adding the sum of the digits after every addition. If the single digit comes out to be 1, then the number is a magic number.

Sample Inputs

Input 1:

 A = 83557

Input 2:

 A = 1291

Expected Outputs

Output 1:

 1

Explanation 1:

 Sum of digits of (83557) = 28
 Sum of digits of (28) = 10
 Sum of digits of (10) = 1. 
 Single digit is 1, so it's a magic number. Return 1.

Output 2:

 0


Explanation 2:

 Sum of digits of (1291) = 13
 Sum of digits of (13) = 4
 Single digit is not 1, so it's not a magic number. Return 0.

Solutions

Basically we return 1 if the sum of digits return 1 else 0

Approach :

  • store s = create a recursive function - return (n%10) + rec(n//10)
  • check s > 9 , if then call again like while s >9:—> s = rec(s)
  • check s is 1 return 1 else return 0

Python :

class Solution:
    # @param A : integer
    # @return an integer
    def solve(self, A):
        def rec(N):
            if N < 10:
                return N
            return (N%10) + rec(N//10)

        s = rec(A)
        
        while s > 9:
            s = rec(s)
        
        if s == 1:
            return 1
        else:
            return 0
