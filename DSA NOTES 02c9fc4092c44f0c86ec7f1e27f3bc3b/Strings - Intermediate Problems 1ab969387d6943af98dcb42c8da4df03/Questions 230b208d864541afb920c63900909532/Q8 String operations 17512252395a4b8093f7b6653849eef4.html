Description

Akash likes playing with strings. One day he thought of applying following operations on the string in the given order:

  • Concatenate the string with itself.
  • Delete all the uppercase letters.
  • Replace each vowel with '#'.

You are given a string A of size N consisting of lowercase and uppercase alphabets. Return the resultant string after applying the above operations.

NOTE: 'a' , 'e' , 'i' , 'o' , 'u' are defined as vowels.

Sample Inputs

Example Input

A="AbcaZeoB"

Expected Outputs

Example Output

"bc###bc###"

First concatenate the string with itself so string A becomes "AbcaZeoBAbcaZeoB".
Delete all the uppercase letters so string A becomes "bcaeobcaeo".
Now replace vowel with '#'.
A becomes "bc###bc###".

Solutions

Basically we need to Delete all the uppercase letters and Now replace vowel with '#'

Approach :

  • create a res array , loop through the given array
  • check if A[i] in 'aeiou': A[i] = '#’ , if so res.append(A[i])
  • check elif ord(A[i]) >= 65 and ord(A[i]) <= 90: if so : A[i] = '’
  • else append A[i] and return res array

Python :

class Solution:
    # @param A : string
    # @return a strings
    def solve(self, A):
        A = list(A)
        res = []
        for i in range(len(A)):
            if A[i] in 'aeiou':
                A[i] = '#'
                res.append(A[i])
            elif ord(A[i]) >= 65 and ord(A[i]) <= 90:
                A[i] = ''
            else:
                res.append(A[i])
        return "".join(res+res)
