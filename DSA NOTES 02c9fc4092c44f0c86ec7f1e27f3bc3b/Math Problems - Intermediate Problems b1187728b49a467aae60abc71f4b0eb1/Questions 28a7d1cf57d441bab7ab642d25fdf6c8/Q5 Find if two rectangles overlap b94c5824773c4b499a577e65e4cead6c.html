Q5.Find if two rectangles overlap

DifficultyEasy
TagMath

Description

Eight integers A ,B,C,D,E,F, and H represent two rectangles in a 2D plane.

For the first rectangle, its bottom left corner is (A, B), and the top right corner is (C, D), and for the second rectangle, its bottom left corner is (E, F), and the top right corner is (G, H).

Find and return whether the two rectangles overlap or not.

Sample Inputs

Input 1:

A = 0   B = 0
C = 4   D = 4
E = 2   F = 2
G = 6   H = 6
 
Input 2:

A = 0   B = 0
C = 4   D = 4
E = 2   F = 2
G = 3   H = 3

Expected Outputs

Output 1:
1

Explanation 1:
Rectangle with bottom left (2, 2) and top right (4, 4) is overlapping.

Output 2:

1

Explanation 2:

Overlapping rectangles can be found.

Solutions

Basically we need to check if two rectangles overlap or not using the given inputs

Approach :

  • check for all possible cases where the rectangle wont overlap
  • if rectangle has area 0, no overlap , If one rectangle is on left side of other
  • If one rectangle is above other

Python

class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y
 
# Returns true if two rectangles(l1, r1)
# and (l2, r2) overlap
def doOverlap(l1, r1, l2, r2):
     
    # if rectangle has area 0, no overlap
    if (l1.x == r1.x || l1.y == r1.y || r2.x == l2.x || l2.y == r2.y)
        return False
     
    # If one rectangle is on left side of other
    if(l1.x > r2.x or l2.x > r1.x):
        return False
 
    # If one rectangle is above other
    if(r1.y > l2.y or r2.y > l1.y):
        return False
 
    return True
 
# Driver Code
if __name__ == "__main__":
    l1 = Point(0, 10)
    r1 = Point(10, 0)
    l2 = Point(5, 5)
    r2 = Point(15, 0)
 
    if(doOverlap(l1, r1, l2, r2)):
        print("Rectangles Overlap")
    else:
        print("Rectangles Don't Overlap")
