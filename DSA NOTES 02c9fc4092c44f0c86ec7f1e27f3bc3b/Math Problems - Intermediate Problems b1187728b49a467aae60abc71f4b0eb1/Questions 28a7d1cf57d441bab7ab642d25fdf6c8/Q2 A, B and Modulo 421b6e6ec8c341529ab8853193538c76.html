Description

Given two integers A and B , find the greatest possible positive integer M, such

that A % M = B % M

Sample Inputs

Input 1:

A = 1
B = 2

Input 2:

A = 5
B = 10

Expected Outputs

Output 1:

1

Output 2:

5

Solutions

Basically we need to return a max integer which is A % M = B % M

Approach :

  • the optimized approach here is to use congruence property
  • (A-B) % M = 0 , i.e (A-B) has to be a multiple of ‘M’. (A-B) = k*M

Now, as we are looking for maximum value of ‘M’.

  • so A - B = M , we need max of M , if A > B :A - B else B - A

notes :

=>  A % M = B % M

=>    A % M - B % M = 0

=>    (A-B) % M = 0

=> i.e (A-B) has to be a multiple of ‘M’.

=> (A-B) = k*M

Now, as we are looking for maximum value of ‘M’.

Therefore, we need to consider the minimum possible value of k.

If k = 1, M = (A-B)
